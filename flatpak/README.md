# ROMM Deck Manager

ROMM Deck Manager is a native Steam Deck Flatpak for browsing and downloading games
from a ROMM server directly into EmuDeck-compatible folders.

## Install
flatpak install flathub com.romm.deck

## Features
- Steam Deck Game Mode UI
- First-run ROMM server login
- Emulator detection and installation
- EmuDeck auto-import
- Download queue with resume
- Steam ROM Manager integration
- Automatic updates via Flatpak

## Notes
Downloads resume automatically if interrupted.
No ROM content is included.
